# PrivateChat / SecureChat Support & Feedback

## Bevorzugter Beta-Kanal

Für TestFlight-Builds soll Feedback direkt über TestFlight gesendet werden:

1. TestFlight öffnen
2. PrivateChat auswählen
3. „Beta-Feedback senden“ verwenden
4. Screenshot und kurze Beschreibung anhängen

## Was Feedback enthalten sollte

```text
Build:
Gerät:
iOS-Version:
Bereich: Onboarding / Relay / Pairing / Chat / Security / Sonstiges

Erwartetes Verhalten:

Tatsächliches Verhalten:

Schritte zum Reproduzieren:
1.
2.
3.
```

Bitte keine Relay-Tokens, privaten Schlüssel oder Chat-Inhalte mitschicken.

## Diagnosebericht

In der App kann unter „Security → Support & Feedback“ ein Diagnosebericht geteilt oder kopiert werden. Der Bericht enthält technische Metadaten, aber keine Chat-Texte, keine privaten Schlüssel und keine Tokens.

## App Store Connect

Zusätzlich muss eine Support-URL in App Store Connect hinterlegt werden. Minimal ausreichend ist eine öffentliche Support-Seite mit den obigen Hinweisen oder eine Projekt-/Kontaktseite.
